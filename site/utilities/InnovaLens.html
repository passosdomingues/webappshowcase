Sobre o InnovaLens

O InnovaLens é uma ferramenta de análise semântica de patentes que utiliza processamento de linguagem natural para extrair conceitos-chave, componentes, funcionalidades e aplicações de documentos de patentes.

Funcionalidades

  • Análise semântica de textos de patentes
  • Extração automática de conceitos e componentes
  • Geração de queries para busca no Espacenet
  • Visualização de relacionamentos entre conceitos
  • Dicionários personalizáveis
  • Extração automática de termos do texto
  • Interface responsiva com tema claro/escuro
