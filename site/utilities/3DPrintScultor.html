Ajuda & Atalhos de Teclado

Fluxo de Trabalho:

  1. Carregue uma imagem (PNG, JPG, BMP, TIFF)
  2. Ajuste os parâmetros de segmentação conforme necessário
  3. Gere o modelo 3D a partir da imagem processada
  4. Exporte o modelo como STL se desejar
  5. Fatie o modelo para impressão 3D
  6. Gere o G-code final para sua GTMax H5

Atalhos de Teclado:

  • R: Resetar visualização da câmera
  • W: Alternar visualização wireframe
  • F: Focar no modelo
  • H: Mostrar/ocultar ajuda
  • 1-5: Executar etapas 1-5 do pipeline

Dicas:

  • Imagens com bom contraste produzem melhores resultados
  • Ajuste a suavização para reduzir artefatos no modelo 3D
  • Use a otimização para impressão 3D para melhorar a qualidade de impressão
  • O preenchimento em hexágono com 5% é recomendado para a maioria dos modelos
