Política de Privacidade e Termos de Uso

Conformidade com a Lei Geral de Proteção de Dados (LGPD)

1. Finalidade do Sistema: Este sistema tem como objetivo registrar visitas domiciliares do CRAS para fins de assistência social, acompanhamento familiar e elaboração de políticas públicas.

2. Dados Coletados: Coletamos apenas dados necessários para a prestação de serviços sociais, incluindo:

  • Dados de identificação da família
  • Informações socioeconômicas
  • Condições de saúde e vulnerabilidade
  • Localização geográfica para planejamento territorial

3. Armazenamento e Segurança: Os dados são armazenados localmente no dispositivo do usuário, com criptografia e controle de acesso. Backup periódico é realizado para preservação das informações.

4. Compartilhamento: Os dados só serão compartilhados com órgãos públicos competentes para fins de políticas públicas, respeitando o sigilo e a privacidade.

5. Direitos do Titular: As famílias têm direito a:

  • Acesso aos seus dados
  • Correção de informações inexatas
  • Exclusão de dados (exceto quando obrigatório por lei)
  • Revogação do consentimento

6. Isenção de Responsabilidade: O sistema é uma ferramenta de apoio à gestão social. A qualidade e veracidade dos dados inseridos são de responsabilidade do usuário. O desenvolvedor não se responsabiliza por:

  • Perda de dados devido a falhas técnicas do dispositivo
  • Uso indevido das informações por terceiros
  • Decisões tomadas com base nos dados do sistema

7. Contato do Encarregado (DPO): Para questões sobre proteção de dados, entre em contato com o CRAS de Fama-MG.
